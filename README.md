# 🌙 AstroCalc Multi-Tradisi

Aplikasi astrologi open-source yang menggabungkan tiga tradisi:
- **Astrologi Barat** (Tropis, Placidus houses)
- **Ba Zi** (Four Pillars of Destiny)
- **Primbon Jawa** (Wetonan)

## 🎯 Filosofi

> "Ikhtiar mengenal diri, bukan prediksi takdir"

Aplikasi ini dirancang sebagai alat **tafakkur** dan **refleksi**, dengan filter teologis Islam untuk memastikan penggunaan yang etis.

## 🚀 Deploy ke GitHub Pages

1. Fork repository ini
2. Enable GitHub Pages di Settings > Pages
3. Pilih source: `main` branch, folder `/ (root)`
4. Akses di `https://username.github.io/astro-calc/`

## 📱 Fitur

- ✅ Perhitungan natal chart lengkap
- ✅ Ba Zi 4 pilar dengan analisis elemen
- ✅ Weton Jawa dengan primbon
- ✅ Konvergensi multi-tradisi
- ✅ Filter teologis Qur'ani
- ✅ Transit 2025-2026
- ✅ Export JSON & PDF
- ✅ PWA (Install di Android/iOS)
- ✅ Offline capable

## ⚠️ Disclaimer

- Perhitungan planet menggunakan algoritma simplifikasi (±1° akurasi)
- Untuk perhitungan presisi tinggi, gunakan Swiss Ephemeris
- Ba Zi menggunakan konversi lunar approximation
- Primbon Jawa adalah tradisi budaya, bukan ilmu pasti
